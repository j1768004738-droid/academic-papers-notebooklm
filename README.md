# 学术论文聚合库

为 Google NotebookLM 优化的学术论文集合。

## 📚 关于本库

本库包含 **54** 个学术论文和书籍，已清理文本格式，可直接导入 Google NotebookLM 进行分析和研究。

### 特点
- ✓ 清理后的文本内容（移除页眉、页脚、参考文献）
- ✓ UTF-8 编码，支持中文
- ✓ 静态 HTML 格式，无需 JavaScript
- ✓ 适合 NotebookLM 爬虫抓取

## 📑 批次文件

- [Collection_Batch_01.html](Collection_Batch_01.html) - 第一批（40 篇）
- [Collection_Batch_02.html](Collection_Batch_02.html) - 第二批（14 篇）

## 🚀 如何使用

1. 打开 [Google NotebookLM](https://notebooklm.google.com)
2. 创建新的 Notebook
3. 点击"添加源"
4. 选择"Web 链接"
5. 粘贴下方任意批次的完整 URL
6. NotebookLM 将自动抓取并分析内容

- `Collection_Batch_01.html` - 第一批论文（40 篇）
